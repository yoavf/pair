What happens when you put two coding agents side by side?

Pair(claude) runs two Claude Code sessions in parallel, collaborating on the same task. One acts as Navigator, planning and reviewing, while the other takes the role of Driver, writing the code.

FAQ

What is Pair(claude)?

Pair(claude) is an experiment in running two Claude Code sessions in pair programming mode.

How does it work?

Using the Claude Code sdk and two sessions, messages are relayed between a Driver who makes the change and a Navigator who reviews and provides guidance and feedback. The navigator approves all file edits, and when asked by the Driver, performs a full code review.

How can I try this?

Run npx @yoavf/pair claude to start a Pair(claude) session.

If you want the pair command available permanently on your system, install it globally with npm install -g @yoavf/pair. After that, you can run pair claude anytime.
